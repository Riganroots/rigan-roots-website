Google Sheet CRM Setup:
1. Create a Google Sheet with columns: Date, Name, Phone, Email, Destination, Travelers, Duration, Hotel, Budget, Travel Month, Visa Type, Contact Method, Estimated Price, Message.
2. Open Extensions → Apps Script and deploy a Web App.
3. Paste your Web App URL inside the JavaScript variable GOOGLE_SCRIPT_URL below.
4. Until you add the URL, the page will still send leads through WhatsApp and copy details.
